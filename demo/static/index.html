Concurrent FIFO Text Editor

Live collaboration, version conflicts, and shared state in motion.

This demo sits on top of the real C server and visualises authentication, role-based access, optimistic concurrency control, and multi-client updates.

How To Demo It

Use two writers, then show one reader seeing the same shared state

1. Connect the users

Click Connect Demo Users. This opens real FIFO client sessions.

2. Edit as the writers

daniel and yao can both write. Use them for insert, delete, and formatting commands.

3. Show collaboration

Switch to ryan and use get or Refresh Snapshot to show the read-only user seeing the latest state.

4. Show concurrency

Run Stale Write Demo to show one writer succeeding, the version incrementing, and the other writer getting STALE_VERSION.
